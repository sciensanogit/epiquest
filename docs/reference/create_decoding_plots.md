# Visualize decoded hidden states

Generates two plots to visualize state assignments for (out-of-sample)
state decoding: one with solid colored points (fixed state assignment)
and another where point transparency (alpha) is weighted by the decoding
probability. The data can be *out-of-sample*, meaning that they are new
data on which the hidden Markov model was not trained.

## Usage

``` r
create_decoding_plots(list_results, method = "filtering", print = FALSE)
```

## Arguments

- list_results:

  An object of class `epiquest_decoding` produced by
  [`run_out_of_sample_decoding()`](https://sciensanogit.github.io/epiquest/reference/run_out_of_sample_decoding.md).

- method:

  A character string specifying the decoding method to visualize. Must
  be one of `'filtering'` (default), `'local'`, or `'global'`.

- print:

  A logical. If `TRUE`, all generated plots are printed to the active
  graphics device.

## Value

An object of class `epiquest_plot_list`, a named `list` of `ggplot2`
objects containing:

- `time_series`: A ggplot object with points colored by the most likely
  state.

- `time_series_shaded`: A ggplot object similar to `time_series`, but
  where point transparency represents state probability (not produced if
  `method = 'global'`).

## Details

Consult
[`run_out_of_sample_decoding`](https://sciensanogit.github.io/epiquest/reference/run_out_of_sample_decoding.md)
for information about the different decoding options for method. The
vertical dotted line marks the last index of the training data for
`hmm_frozen`. Data points to its right are the new out-of-sample data.

The plots generated by
[`create_hmm_plots()`](https://sciensanogit.github.io/epiquest/reference/create_hmm_plots.md)
and
[`create_loop_plots()`](https://sciensanogit.github.io/epiquest/reference/create_loop_plots.md)
all use `'local'` state assignments.
