# Summary of QUEST thresholds

Provides a readable summary of the calculated thresholds and the HMM
configuration used to derive them.

## Usage

``` r
# S3 method for class 'epiquest_thresholds'
summary(object, ...)
```

## Arguments

- object:

  An object of class `epiquest_thresholds`.

- ...:

  Additional arguments passed to methods.

## Value

The function prints a summary to the console and invisibly returns the
`object`.
