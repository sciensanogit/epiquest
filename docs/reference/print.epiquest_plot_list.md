# Print method for EpiQUEST plot lists

Ensures that when a list of plots is returned to the console, the plots
are drawn automatically without printing the list structure.

## Usage

``` r
# S3 method for class 'epiquest_plot_list'
print(x, ...)
```

## Arguments

- x:

  An object of class `epiquest_plot_list`.

- ...:

  Additional arguments passed to methods.
